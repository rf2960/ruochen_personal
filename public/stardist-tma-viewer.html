Biomedical AI / whole-slide image analysis

StarDist nuclear segmentation viewer.

Scale-aware H&E tissue microarray analysis pipeline using OpenSlide, TensorFlow, and the pretrained StarDist 2D_versatile_he model. The workflow normalizes 40x SVS tiles to model scale, applies stain-gated filtering, deduplicates overlapping tile detections, and exports self-contained HTML viewers for core-level and patch-level review.

The full viewers are hosted as GitHub Release assets because they are large, self-contained HTML files with embedded image tiles.

Pipeline

What the system does

WSI ingestion

Reads 40x SVS slides with OpenSlide, detects seven TMA cores from a low-resolution overview, and preserves spatial context for downstream review.

Deep-learning segmentation

Runs pretrained StarDist on scale-normalized H&E tiles, with per-core thresholds and nuclear mask-size filtering to improve practical sensitivity.

Quality review

Exports interactive HTML viewers with core navigation, spatial tile grids, high-resolution patch popups, and boundary opacity controls.
